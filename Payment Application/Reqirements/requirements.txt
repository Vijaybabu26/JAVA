CLI Payments methods : - 
1. Can Create Account & Self register
2. Adding Bank Account
3.Adding money to wallet (Bank account to wallet)
4.Send money/ first transaction.
	Source   ---> Destination
	BankAcc---> Wallet
	Wallet   ---> BankAcc
	BankAcc + Wallet ---> Wallet
5. Statements (Account Transaction statements).
6.Refund.
7.Monthly Transactions.

	1. user can create account /can self register (phno, name, password, dob, addr)
	2. user should login to application to perform any designated task
	3. user can add bankaccount  (optional, multiple BA) 
	4. user can add money to wallet ( limit of wallet amount)
	5. user can do a txn (txn limit, source to dest, )
	6. user can request for a statement (txn stmt, limit of txns req)
	7. user can request for a refund ()
	8. monthly statement generation
